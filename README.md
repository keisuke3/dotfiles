# dotfiles

## Installation

1. Run the following command in your terminal:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/keisuke3/dotfiles/main/init.sh)"
   ```
2. Change to the dotfiles directory:
   ```bash
	cd dotfiles
   ```
3. Run the installation script:
   ```bash
   make all
    ```
